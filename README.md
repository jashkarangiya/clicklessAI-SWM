# ClicklessAI — Semantic Web Mining Project

AI-powered system that automates web interactions using LangGraph orchestration, Playwright scraping, and a modern frontend.

## Project Structure

```
clicklessAI-SWM/
├── docs/           → Documentation (shared)
├── frontend/       → UI / Frontend
├── scraper/        → Playwright scraper
├── orchestration/  → LangGraph orchestration
├── backend/        → API server
├── database/       → Database layer
├── README.md
├── CONTRIBUTING.md
└── RULES.md
```

## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/jashkarangiya/clicklessAI-SWM.git
   ```
2. Create a branch from `main`
   ```bash
   git checkout -b feature/your-feature
   ```
3. Make changes, commit, push, and open a Pull Request

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full workflow and [RULES.md](RULES.md) for project rules.
